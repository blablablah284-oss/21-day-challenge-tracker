# 🔥 21 DAY MASTER CHALLENGE TRACKER

An ultra-modern, futuristic web app to track your 21-day transformation journey with 4 daily missions:

1. **NO FAP** 🔥 - Control your mind with motivational quotes
2. **BETTER SLEEP** 😴 - Track sleep hours and quality
3. **NO UNHEALTHY FOOD** 🥗 - Log all food items
4. **STUDY** 📚 - Even 1 second counts!

## 🚀 Features

✨ **Futuristic UI** - Glassmorphism, neon accents, smooth animations
✅ **Real-time Countdown** - Challenge timer starts immediately
📅 **Interactive Calendar** - View all 21 days with progress indicators
💾 **Local Storage** - Your data persists across sessions
🎯 **Mission Tracking** - Complete or fail with reasons
📊 **Progress Analytics** - Track your journey
🎨 **Modern Design** - Dark theme, responsive layout

## 📦 Installation

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## 🛠️ Tech Stack

- **React 18** - UI Library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Date-fns** - Date utilities

## 📱 Responsive

Works flawlessly on mobile, tablet, and desktop devices.
